@index() Alias @i()

不传参可省略括号

:doodle { @grid: 10 / 20vw; grid-gap: 1px; } background: teal; :after { content: @i; color: white; }

example

:doodle { @grid: 10x1/ 20vw; } @place-cell: center; @size: calc(100%/10*@i); opacity: calc(1.1 - 1 / 10 * @index); background: teal;

@row @col

:doodle { @grid: 5 / 20vw; grid-gap: 1px; } background: teal; :after { content: (@row,@col); color: white; }

@size-row 行数

:doodle { @grid: 5 / 20vw; grid-gap: 1px; } background: teal; :after { content: @row / @size-row; color: white; }

@size-col 列数

:doodle { @grid: 5 / 20vw; grid-gap: 1px; } background: teal; :after { content: @col / @size-col; color: white; }

@size 格数

:doodle { @grid: 5 / 20vw; grid-gap: 1px; } background: teal; :after { content: @i / @size; color: white; }

example

:doodle { @grid: 5 / 20vw; grid-gap: 1px; }; background: rgba( 0, 128, 128, calc(@row * @col / @size));

@pick(v1, v2,...) Alias @p

:doodle { @grid: 5 / 20vw; grid-gap: 1px; }; background: teal; opacity: @pick(1, .6, .3, .1); :after { content: @pick(1, 2, 3, 4, @i); color: white; }

@pick-n(v1, v2,...) Alias @pn

顺序选取

:doodle { @grid: 5 / 20vw; grid-gap: 1px; }; background: teal; opacity: @pick-n(1, .6, .3, .1); :after { content: @pick-n(1, 2, 3, 4); color: white; }

@pick-d(v1, v2,...) Alias @pd

和@pick-n类似 但有一定的随机性

:doodle { @grid: 5 / 20vw; grid-gap: 1px; }; background: teal; opacity: @pick-d(1, .6, .3, .1); :after { content: @pick-d(1, 2, 3, 4); color: white; }

@rand(start [,end]) Alias @r

取随机值 @rand(a, z); @rand(.4em, 1.2em); etc...

:doodle { @grid: 5 / 20vw; } background: rgba(0, 128, 128, @rand(.9)); transition: .2s ease @rand(200ms); will-change: transform; transform: rotate(@rand(360deg)); clip-path: polygon( @rand(100%) 0, 100% @rand(100%), 0 @rand(100%) );
