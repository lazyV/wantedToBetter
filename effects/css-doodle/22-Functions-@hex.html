@hex(num)

将数字转换为十六进制格式。

:doodle { @grid: 9x9; @size: 30vw; } :after { content: \@hex(@rand(9527, 9687)); color: teal; font-size: 1.2em; }


example

:doodle { @grid: 20 / 100vmax; background: black; font-family: sans-serif; overflow:　hidden; } :after { content: \@hex(@r(0x2500, 0x257f)); color: hsla(@r360, 70%, 70%, @r.9); font-size: 5vmax; }
